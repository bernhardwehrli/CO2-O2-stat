# CO2-O2-stat
Mathematica notebook for calculating covariance parameters and average 24-h cyles form concentration timeseries of dissolved CO2-O2 pairs in river water. 
---------------------------------------------------------------------------------------------------------------------------------------------------------
CO2-O2-stat-v1.nb is the Mathematica noteboook written in Mathematica 14.3  by Bernhard Wehrli;  version 1.0 Nov 18 2025

Balanova_t_exCO2_exO2_2018.txt is a TAB delimited textfile with 3 columns: date-time, excess CO2 (mikromolar), excess O2 (mikromolar). 
Data were acquired at 15' time intervalls at the Balanaova station in the Danube Delta. The text file is used as input-data-file in the example notebook.
